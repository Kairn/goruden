# N3-N1の文法

## というと / といえば

* すぐ思いつく
  * N＋というと～
    * ベトナム料理**といえば**フォーですね。
    * 日本**といえば**桜ですね。
* ～だけと
  * 普通形＋といえば＋普通形＋が、～～
  * 弱い肯定＋弱い否定
    * 彼女は静かだ**といえば**静かだが、家ではよく話す。
    * 今日は寒い**といえば**寒いけど、コートを着るほどじゃないね。

## といった
* ～のようだ
  * N1、N2＋といった＋N
  * いくつかの例をあげ、それをまとめる時に使う
    * ドイツでも寿司やラーメン**といった**日本料理が最近人気だ。
    * トヨタやホンダ**といった**日本企業は世界でも有名だ。
    * 夏の楽しみといえば、音楽フェス、花火大会、お祭り**といった**大規模なイベントを思い浮かぶだろう。
    * 日本の食卓に欠かせない、みそやしょうゆ**といった**調味料の原料となる大豆は、日本国内で作られてるのはわずかで、その多くは海外からの輸入に頼っています。

## としたら / とすれば
* もし～たら
  * 普通形＋としたら
  * 非現実的な状況も含む
    * 仮に通勤に往復二時間かける**とすると**、一年で２４０時間以上も通勤しているということになる。
    * もし、私の言ったことに何か失礼があった**としたら**深くお詫びします。

## として
* という立場で
  * N＋として
    * 子供を応援するのは親**として**当然だ。
    * 田中さんは有名な歌手でうが、妻**としても**母**としても**素晴らしいです。
    * 例え今回不合格だった**としても**、今までの君の努力が無駄になる訳ではない。
    * それは意見**としては**いいですが、本当に実現可能か、検証する必要があります。

## によって / によっては
* それぞれ違う
  * 日本人も人**にとって**性格が様々だ。
  * スーパー**によっては**、２４時間営業のところもある。

## によると / によれば
* 伝言
  * 今日の星占い**によると**、いて座の運勢は最高だそうです。
  * 政府関係者の話**によれば**近いうち、消費税が８％から１０％に上がるらしい。

## よって / より / よる
* 原因と結果
  * 最近、バイク**による**交通事故が増えている。
  * 北極の氷を調べること**によって**、数十万年前の地球の気候を知ることができた。
  * 失敗すること**によって**成功に近づくことができる。

## よらず
* 関係ない
  * あの人は見かけに**よらず**優しい。
  * 買い物などの日常生活における決済手段は、支払い金額に**よらず**現金の使用は減少傾向で、クレジットカードや電子マネーなどのキャッシュレス決済の利用は増加傾向にあります。
