# N3-N1の文法

## というと / といえば

* すぐ思いつく
  * N＋というと～
    * ベトナム料理**といえば**フォーですね。
    * 日本**といえば**桜ですね。
* ～だけと
  * 普通形＋といえば＋普通形＋が、～～
  * 弱い肯定＋弱い否定
    * 彼女は静かだ**といえば**静かだが、家ではよく話す。
    * 今日は寒い**といえば**寒いけど、コートを着るほどじゃないね。

## といった
* ～のようだ
  * N1、N2＋といった＋N
  * いくつかの例をあげ、それをまとめる時に使う
    * ドイツでも寿司やラーメン**といった**日本料理が最近人気だ。
    * トヨタやホンダ**といった**日本企業は世界でも有名だ。
    * 夏の楽しみといえば、音楽フェス、花火大会、お祭り**といった**大規模なイベントを思い浮かぶだろう。
    * 日本の食卓に欠かせない、みそやしょうゆ**といった**調味料の原料となる大豆は、日本国内で作られてるのはわずかで、その多くは海外からの輸入に頼っています。

## としたら / とすれば
* もし～たら
  * 普通形＋としたら
  * 非現実的な状況も含む
    * 仮に通勤に往復二時間かけるとすると、一年で２４０時間以上も通勤しているということになる。
    * もし、私の言ったことに何か失礼があったとしたら深くお詫びします。

## として
* という立場で
  * N＋として
    * 子供を応援するのは親として当然だ。
    * 田中さんは有名な歌手でうが、妻としても母としても素晴らしいです。
    * 例え今回不合格だったとしても、今までの君の努力が無駄になる訳ではない。
    * それは意見としてはいいですが、本当に実現可能か、検証する必要があります。
