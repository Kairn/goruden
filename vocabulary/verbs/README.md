# 《黄金》- 单词篇 - 动词部分

本文档主要记录日语动词和其常用形态以及变形规则。由于日语动词的变形实在太多，在单词列表里面无法一一枚举，所以只挑选了有代表性的形态。同样的，动词的声调也不一一标注。

## 动词三分类
日语当中所有的动词原形（辞书形）都是以う段的音结尾的。下面分类以及变形规则全部会以原形为基础。

### 一类动词
1. 又称五段动词。
2. 如果结尾是除了る以外的う段，那必定是一类。
3. 如果结尾是あ段+る、う段+る、或者お段+る， 那么也属于一类。
4. 有少数不符合上述规则的特殊动词也属于一类（参考下面特例）。

#### 特例（不一定全）
* 切る
* 帰る
* 走る
* 入る
* 気に入る
* 滑る
* 知る
* 要る
* 減る

### 二类动词
1. 如果是一类动词的特例，则不会属于二类。
2. 除了特例，如果结尾是い段+る，则属于二类动词中的上一段动词。
3. 除了特例，如果结尾是え段+る，则属于二类动词中的下一段动词。
4. 上、下段动词在变形时没有实质的区别，只是古日语中遗留的一种分类。

### 三类动词
1. 又称不规则动词，变形时的规则会比较特殊（有一种随机的感觉，通常只能死记）。
2. 来る（没错只有这一个词）属于三类动词中的カ变动词。
3. する，或者名词+する属于三类动词中的サ变动词。
4. カ变、サ变动词在变形时会有不一样的规则。

## 自动词和他动词

### 区分方法
1. 如果结尾是す，基本上都属于他动词。
2. 如果结尾是え段+る，而且该词没有以す结尾的动词，那么这个词也很可能属于他动词。而对应的自动词很可能是あ段+る。
3. 如果根据以上规律判断不出是他动词，那很有可能是自动词，但是也有不少例外。

### 自动词
* 使用自动词主要是描述一种状态或现象，说话人不想强调这个现象是因什么引起的。
* 经常以<名词>+が+<自动词>的形式出现。
* 在名词表示经过的场所或离开的地点时，（移动性）自动词前面也用を做助词。
* 移动性动词基本都属于自动词。
* 同族目的语（名词和动词具有相同词根）可以使用を做助词，例如 `踊りを踊る`。

### 他动词
* 使用他动词会强调一种动作或行为，说话人相对在意是什么引发了某种事件。
* 经常以<名词>+を+<他动词>的形式出现。

## 变形规则
某些变形可能涵盖了其他的变形作为基础，所以规则当中要参照着看。

### 连用形
1. 一类动词将结尾的う段变为同行的い段。
2. 二类动词直接将结尾的る去掉。
3. 来(く)る -> 来(き)。
4. する -> し。

### ます形
将原形变为连用形之后加上ます。

#### 再变形
1. ません，表示否定。
2. ました，过去式。
3. ませんでした，过去否定。

### て形
1. 一类动词
   * 结尾是す的话将其变为して。
   * 结尾是く的话将其变为いて。
   * 结尾是ぐ的话将其变为いで。
   * 结尾是う、つ、る的话将其变为って。
   * 结尾是ぶ、む、ぬ的话将其变为んで。
   * 行く -> 行って，这是特例。
2. 二类动词直接把结尾的る变成て。
3. 来る -> 来(き)て。
4. する -> して。

### た形
和て形一样的规则，只需要把结尾的て或で改成た或だ（清音对清音，浊音对浊音）。

### 未然形
1. 一类动词将结尾的う段变为同行的あ段，但是う本身会变成わ而不是あ。
2. 二类动词直接将结尾的る去掉。
3. 来(く)る -> 来(こ)。
4. する多数情况变为し，但是也可能变为せ或さ。

### ない形
将原形变为未然形之后加上ない。但是ある会直接变为ない。

### 命令形
1. 一类动词将结尾的う段变为同行的え段。
2. 二类动词将结尾的る变成ろ。
3. 来る -> 来(こ)い。
4. する -> しろ。

### 意志形
1. 一类动词将结尾的う段变为同行的お段+う（相当于长音）。
2. 二类动词将结尾的る变成よう。
3. 来る -> 来(こ)よう。
4. する -> しよう。
5. ます -> ましょう。

### 假定形（ば形）
1. 一类动词将结尾的う段变为同行的え段再加上ば。
2. 二类动词将结尾的る变成れば。
3. 来る -> 来（く）れば。
4. する -> すれば。

### 可能形
1. 一类动词将结尾的う段变为同行的え段再加上る。
2. 二类动词将结尾的る变成られる。
3. 来る -> 来(こ)られる。
4. する -> できる。
5. 这个形式可以作为全新的二类动词。
6. 日本现在也出现了把ら去掉的形式。

### 被动形
1. 一类动词变成未然形后加上れる。
2. 二类动词将结尾的る变成られる。
3. 来る -> 来(こ)られる。
4. する -> される。
5. 这个形式可以作为全新的二类动词。

### 使役形
1. 一类动词变成未然形后加上せる。
2. 二类动词将结尾的る变成させる。
3. 来る -> 来(こ)させる。
4. する -> させる。
5. 这个形式可以作为全新的二类动词。

### 使役被动形
1. 一类动词变成未然形后加上せられる，日语中尝尝把せら约音成さ。
2. 二类动词将结尾的る变成させられる。
3. 来る -> 来(こ)させられる。
4. する -> させられる。
5. 这个形式可以作为全新的二类动词。
